This section describes how to install the xraybinaryorbit package.

To install xraybinaryorbit from the Python Package Index (PyPI), you can use pip, the Python package installer. Run the following command in your terminal or command prompt:

pip install xraybinaryorbit

This will automatically install the latest stable version of xraybinaryorbit along with its dependencies.

If you prefer to install the package from source, you can clone the repository from GitHub and install it manually:

  1. Clone the repository:

    git clone https://github.com/xragua/xraybinaryorbit

  2. Navigate to the package directory:

    cd xraybinaryorbit

  3. Install the package using pip:

    pip install .

xraybinaryorbit requires the following Python libraries:

  • numpy

  • pandas

  • scipy

  • pyswarm

  • matplotlib

  • os

  • warnings

  • tkinter

  • astropy

  • inspect

These dependencies are automatically installed when you install the package via pip. If you encounter any issues, ensure that these libraries are installed on your system.
